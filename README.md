### How to Run Locally

```bash
# Clone the repo
git clone https://gitlab.com/snoog1/misha-website.git
cd misha-website

# Install dependencies
npm install

# Start development server (view on phone too!)
npm run dev -- --host

# Build for production
npm run build

# Preview production build
npm run preview -- --host

Open http://localhost:5173 (or your phone’s network IP) and enjoy!
